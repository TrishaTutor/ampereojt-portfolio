# Ampere OJT Portfolio Website

## INSTALLATION PROCESS

STEP 1: INSTALL GIT
1. Open your browser
2. Go to https://git-scm.com
3. Download and install Git
4. Use default settings

Verify installation:
git --version

-----------------------------------

STEP 2: CREATE A GITHUB ACCOUNT
1. Go to https://github.com
2. Sign up for a free account
3. Log in to GitHub

-----------------------------------

STEP 3: DOWNLOAD THE PROJECT
1. Open the project repository on GitHub
2. Click the green "Code" button
3. Select "Download ZIP"
4. Extract the ZIP file

-----------------------------------

STEP 4: OPEN THE PROJECT FOLDER
1. Open Terminal / Command Prompt
2. Navigate to the project folder:

Windows:
cd path\to\project-folder

macOS / Linux:
cd path/to/project-folder

-----------------------------------

STEP 5: CONFIGURE GIT (FIRST TIME ONLY)

git config --global user.name "Your Name"
git config --global user.email "your@email.com"

Check configuration:
git config --list

-----------------------------------

STEP 6: INITIALIZE GIT (IF REQUIRED)

git init

-----------------------------------

STEP 7: OPEN PROJECT FILES
1. Open the folder in a code editor (VS Code recommended)
2. Read the README.md file
3. Start practicing Git commands

-----------------------------------
